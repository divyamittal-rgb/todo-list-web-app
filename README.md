# To-Do List Web Application

## 📌 Overview
A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript. This project allows users to add, delete, and mark tasks as completed.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and clean user interface

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation & Event Handling)

## 📚 What I Learned
- DOM manipulation
- Event handling in JavaScript
- Structuring a basic web project
- Connecting HTML, CSS, and JS files

## 🎯 Future Improvements
- Add local storage for saving tasks
- Improve UI design
- Add task counter
